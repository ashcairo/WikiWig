FEHLERBEHEBUNG BEI DER INSTALLATION

- Um den Fehler "Konnte Datei/Verzeichnis nicht speichern/anlegen" zu beheben

müssen Sie die Berechtigungen der in der Fehlermeldung aufgeführten Dateien oder Ordner ändern.
Verbinden Sie sich mit Ihrem Server über einen SSH Client oder mit Filezilla und ändern Sie die Berechtigungen der betreffenden Dateien und Verzeichnisse.

(Sie können auch die Rechte des gesamten wikiwig-Verzeichnisses rekursiv ändern, z.B. so:
am ssh Prompt eingeben:
chmod -R 777 wikiwig
)

Danach setzen Sie die Installation wie gewohnt fort.
