RESOLUTION DES PROBLEMES D'INSTALLATION

- Resoudre "Impossible de créer/d'ecrire sur le fichier/le dossier"

Vous devez changer les droits sur le fichier/dossier dont il est question dans le message d'erreur
Pour cela connectez vous au serveur avec un client SSH ou filezilla et changez les droits sur les fichiers/dossiers

(Vous pouvez changer les droits sur le repertoire wikiwig et tous ses sous dossiers par exemple:
a l'invite ssh tapez :
chmod -R 777 wikiwig
)

Reprendre ensuite l'installation
